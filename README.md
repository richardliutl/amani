# Amani

Amani is your friendly neighborhood virtual kitchen manager.

## Requirements
```
npm 7.3.0
node 15.5.1
```

## Getting Started
```bash
npm install
npm start
```

Then open in your browser [http://localhost:3000](http://localhost:3000)
